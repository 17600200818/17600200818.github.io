restful

api的restful设计原则

crud和get post put delete对应关系

c =》 get 查
r =》 post 增
u =》 put 更新
d =》 delete 删除

例子

创建用户

/user method post
201 创建成功
400 请求错误
500 服务器内部错误

用户登陆

/user/:username post
200 ok
400 请求错误
500 服务器内部错误

获取用户信息

/user/:username get
200 ok
400 请求错误
401 用户认证失败
403 用户认证成功但是没有权限
500 服务器内部错误

注销用户

/user/:username delete
204 删除成功
400 请求错误
401 用户认证失败
403 用户认证成功但是没有权限
500 服务器内部错误
